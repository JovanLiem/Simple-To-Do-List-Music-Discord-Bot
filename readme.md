# To Do List Bot

## Commands to add list / create / delete
| Command                  | Deskripsi                                            | Contoh                       |
| ------------------------ | ---------------------------------------------------- | ---------------------------- |
| `!add [tanggal] <tugas>` | Tambah tugas di tanggal tertentu (default: hari ini) | `!add 2025-11-09 Belajar Go` |
| `!list [tanggal]`        | Lihat daftar tugas di tanggal tertentu               | `!list 2025-11-08`           |
| `!done <tanggal> <no>`   | Tandai tugas selesai                                 | `!done 2025-11-09 1`         |
| `!delete <tanggal> <no>` | Hapus tugas tertentu                                 | `!delete 2025-11-08 2`       |
| `!clear [tanggal]`       | Hapus semua tugas di tanggal tertentu                | `!clear 2025-11-09`          |
| `!dates`                 | Lihat semua tanggal yang punya tugas                 | `!dates`                     |
